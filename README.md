Riddle Master – Brain Teaser Web App

---> Project Overview

Riddle Master is a JavaScript web application that allows users to explore and solve riddles. The application fetches riddles from a public API and displays them dynamically on the webpage.

Users can search riddles, filter by category, reveal answers, and save their favorite riddles.

This project demonstrates JavaScript concepts such as API integration, DOM manipulation, and array higher-order functions.

---

---> API Used

This project uses the Riddles API.

Base URL:
https://riddlesapidoc.vercel.app

The API returns riddles with the following properties:

- riddle
- answer
- category

---

---> Features

The application will include the following features:

- Display random riddles
- Reveal answer functionality
- Search riddles by keywords
- Filter riddles by category
- Sort riddles alphabetically
- Save favorite riddles
- Responsive design
